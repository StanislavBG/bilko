# Feature Rules

This directory contains rules specific to individual features as they are built.

## Current Features

- None yet (Phase 1 is foundation only)

## Rule Template for New Features

When adding a new feature, create a rule file with:

```markdown
# Feature: [Feature Name]

Rule ID: FEAT-XXX
Priority: [CRITICAL|HIGH|MEDIUM|LOW]
Version: 1.0.0

## Context
[When this feature rule applies]

## Directives
[Specific implementation requirements]

## Dependencies
[Other rules or features this depends on]

## Rationale
[Why these rules exist for this feature]
```
